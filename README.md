# Golden Awards

Projeto gerado com o [Angular CLI](https://github.com/angular/angular-cli) versão 15.0.0.

## Development server

Para subir o servidor, usar o comando `ng serve`. Navegar para `http://localhost:4200/`.

## Build

Para gerar uma build `ng build`. Os arquivos estarão no diretório `dist/`.

## Running unit tests

Para rodar os testes via [Karma](https://karma-runner.github.io) `ng test`.
